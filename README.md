Client-Server-Client WebApp with access security built-in and rapid development velocity in mind - to collect inspiring wisdom from bright minds of today and ages ago.

# Technologies used

Client:
- Rect/Next.js
- Typescript
- Shadcn UI library

Server-Backend:
- FastAPI (REST API)
- SQLAlchemy (ORM layer)
- Pydantic type validation
- SlowApi rate limitting
- Python
- Runs in Docker container

Database Layer:
- PostgreSQL in production
- Runs in Docker container
- SQLite for local, non-dockerized development (that's how I started out, before upgrading to PostgreSQL)
